# MindFeature（マインドフューチャー）

PHP自作
# 概要

人間心理を身体的特徴から理解するために、必要な知識を掲載したサイトを作りました。

知識を掲載できる管理者と閲覧できる一般ユーザーに分けてログインできます。

# 使い方

管理者
